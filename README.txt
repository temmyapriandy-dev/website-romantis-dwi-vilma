WEBSITE ROMANTIS UNTUK DWI VILMA
Dari Timothy Apriandy Putra

Cara membuka:
1. Ekstrak file ZIP.
2. Buka file index.html menggunakan Chrome, Safari, atau browser lain.
3. Website sudah responsive dan nyaman dibuka di HP.

Musik:
- Tombol musik membuka video resmi "Shape of My Heart" dari YouTube.
- Karena kebijakan browser, musik hanya diputar setelah tombol ditekan.
- Koneksi internet dibutuhkan untuk memutar musik.

Cara mengganti teks:
- Buka index.html di VS Code.
- Cari kalimat yang ingin diganti, lalu edit langsung.

Cara memasang online:
- Upload folder ke GitHub.
- Deploy ke Vercel, Netlify, atau GitHub Pages.

Catatan:
- Seluruh tampilan, CSS, dan JavaScript berada dalam satu file index.html.
- Tidak membutuhkan instalasi library tambahan.
