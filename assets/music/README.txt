Letakkan file audio resmi/berlisensi di folder ini dengan nama:

shape-of-my-heart.mp3

File yang dipakai saat ini memiliki kontainer audio MP4 meskipun ekstensi namanya
.mp3. Karena itu, elemen audio pada index.html menggunakan tipe audio/mp4.

Website akan memutarnya sebagai backsound berulang setelah pengunjung menekan
tombol "Putar Lagu Kita". Browser tidak mengizinkan audio bersuara diputar
otomatis sebelum ada interaksi dari pengunjung.
